Pretty Date Filter Display a date with how many seconds, minutes, hours etc.. have passed

Note this filter converts up to 10 weeks into how many minutes, hours, days or weeks, old something is. You can still pass in a date format that will be used should the passed in date be more than 10 weeks ago, we default this to mediumDate.

Filter Example


<p>{{ curly(pretty.dateHours + ' | prettyDate') }}<p>
<p>{{ curly(pretty.dateDays + ' | prettyDate') }}<p>
<p>{{ curly(pretty.dateWeeks + ' | prettyDate') }}<p>
<p>{{ curly(338132700000 + ' | prettyDate') }}<p>
<p>{{ curly(338132700000 + ' | prettyDate: "EEEE, d MMM yyyy \'at\' HH:mm"') }}<p>

Filter Result

{{ pretty.dateHours | prettyDate }}

{{ pretty.dateDays | prettyDate }}

{{ pretty.dateWeeks | prettyDate }}
