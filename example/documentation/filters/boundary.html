Boundary Filter Find the boundary of a list of objects

This currently only supports numbers (integers and floats) to be compared, if working with dates use getMilliseconds() for comparisons.

Filter Example


<p>{{ curly('[4,6,2,7,4,16,9,13,5] | boundary: "max"') }}<p>
<p>{{ curly('[{ value: "6.21" }, { value: "6.04" }, { value: "7.2" }, { value: "6.8" }] | boundary: "min": "value"') }}<p>

Filter Result

{{ [4,6,2,7,4,16,9,13,5] | boundary: "max" }}

{{ [{ value: "6.21" }, { value: "6.04" }, { value: "7.2" }, { value: "6.8" }] | boundary: "min": "value" }}
