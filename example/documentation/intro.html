Angular Toolbelt from Sysen

This toolbelt has been built to fill a gap in common UI or angular service requirements, it is designed to be used with Angular and Bootstrap, however directive templates can be overridden to use other CSS frameworks

View on GitHub

How To: Adding the toolbelt to your project

Note that there are other ways you might include this library however below is the recommended setup.

Downloading the toolbelt

Recommended (production) installation via bower bower install angular-toolbelt --save

Latest (unstable) versions can be downloaded directly from GitHub development or minified distribution files

Incorporating the library

Add the javascript file to your html page normally at the end of the html just before the closing body tag (after the inclusion of angular.js and before your app.js)

<script src="/path/to/angular-toolbelt.min.js"></script>

Update your angular app.js to include the toolbelt

angular.module('yourApp', ['sysen.toolbelt']);

Enjoy the new functionality!
