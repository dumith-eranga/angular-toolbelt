Browser Detection Service Check for browser information and perform actions if required

Service Overview

Service Name
$detectBrowser
Type
Provider
Variables
[String] name
[String] version
Methods
[Boolean] matches(BrowserName[String], VersionNumber[String])
[Boolean] isAllowed()
Config
[DetectBrowserProvider] allow(BrowserName[String], VersionNumber[String])

Current Detected Browsers are (note: not all versions have been tested, however will correctly identify all current ones which have been tested);

  • Internet Explorer
  • Firefox
  • Chrome
  • Safari
  • Opera
If you require support for a specific browser please raise an issue on our GitHub project and we will look into it.

Demo

HTML 

<div data-ng-controller="browserCtrl">
    <p>You are using: {{ curly('browser.name') }} (Version {{ curly('browser.version') }})</p>
    <p data-ng-if="browser.matches('Safari')">Only Safari users can see this message!</p>
    <p data-ng-if="browser.isAllowed()">Our allowed browsers (Chrome ANY, Firefox 30+, Safari 7+ and MSIE 11+) will see this message.</p>
</div>
JavaScript 

angular.module('myApp', [])
    .config(function($detectBrowserProvider) {
        $detectBrowserProvider.allow("Chrome").allow("Firefox", "30").allow("Safari", "7").allow("Internet Explorer", "11");
    });
    .controller('browserCtrl', ['$scope', '$detectBrowser', function($scope, $detectBrowser) {
        $scope.browser = $detectBrowser;
    }]);
Try It

You are using: {{ browser.name }} (Version {{ browser.version }})

Only Safari users can see this message!

Our allowed browsers (Chrome ANY, Firefox 30+, Safari 7+ and MSIE 11+) will see this message.

Tips

You can use the allowed browser state to redirect or display a message to users about possible compatibility issues on your landing page


app.controller('LandingCtrl', ['$location', '$detectBrowser', function ($location, $detectBrowser) {
    if (!$detectBrowser.isApproved()) {
        $location.path('browser-warning');
    }
});
Note: You may want to also use a cookie to store displaying of the message to only show it once.
