Angular Scroll Directive Scroll current page to a location based on ID

Code

In your HTML file
<a data-target="[element]" data-sys-scroll>Link</a>

<section id="[element]"></section>
In your Controller
Nothing required

Tips

This directive also has a listener for events sent to _pageScroll, this means it is possible to automatically scroll the page when it is loaded by adding the following to your controller:
$scope.$emit('_pageScroll', $location.path().substring(1));

Notes

Full directive usage

<a data-target="[element]" data-offset="[integer]" data-sys-scroll="[mode]"></a>

data-target="[element]" refers to the element based on its ID on the page, for example <section id="uniqueElement"></section>

data-offset="[integer]" allow you to specify an offset during the scroll to cater for navigation or padding, should be a number based on the pixel offset required. Default is 0px.

data-sys-scroll="[mode]" currently two modes are supported none (optional parameter) and smooth. These will either jump to or animate the scroll action respectively.

Example

To try this feature use the navigation on this documentation site, or click the two buttons to show a jump or animated scroll to the top of the page
