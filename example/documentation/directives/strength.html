Angular Password Strength Directive Form field strength indicator (for passwords)

Code

In your HTML file

<input type="password" data-ng-model="password" />
<span data-target="password" data-sys-strength></span>

In your Controller

Nothing required

Tips

Combinations of using 'min-length' with 'rank' might result in situations where you cannot reach the required rank with the length of the string, for example a string strength of strong requires a minimum of 7 characters to achieve.

Notes

Full directive usage

<data-target="[element]" data-min-length="[integer]" data-rank="[integer]" data-sys-strength></span>

data-target="[element]" The binding input element (model and name) should match this attribute. This attribute is required to operate the directive

data-min-length="[integer]" The length required before strength checks will be made to update the label. Default is 6.

data-rank="[integer]" What validation 'rank' is required before the password is accepted. Range is 1-6 (Very Weak to Very Strong). Default is 4 (Good).
