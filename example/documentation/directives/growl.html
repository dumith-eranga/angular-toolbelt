Angular Growl Directive Display notifications to users

Code

In your HTML file

<div data-timeout="5" data-sys-growl="3"></div>
Note: data-timeout attribute is optional and used to provide a number of seconds to display items for, default is forever

In your Controller

$scope.$emit('_addGrowl', messageObject);

Tips

Add a CSS class to the HTML div and position your notifications on the page in a prominent location, e.g. 
<div class="growl" data-sys-growl></div>

.growl {
  position: fixed;
  width: 400px;
  top: 80px;
  right: 40px;
  z-index: 1000;
}

The growl directive also broadcasts a _removeGrowl message when closing the growl containing the closed growls message object, this can be used to then clear a list you might be holding of current growls (e.g. In a cookie). 

scope.$on('_removeGrowl', function(message) { ... });

Notes

The contents of messageObject in the example, used to $emit the message should be the following attributes:

type: string

Values of string should relate to bootstrap alert types; info (default), success, warning, danger

title: string

Any plain text title to display in the growl alert

content: HTML

HTML content top be used in the alert

e.g.

var messageObject = {
      type: 'info',
      title: 'Example Growl',
      content: 'My <em>message</em>'
    };
