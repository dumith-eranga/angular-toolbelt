Angular Growl Directive Display notifications to users

Code

In your HTML file
<div data-sys-growl="3"></div>
In your Controller
$scope.$emit('_addGrowl', messageObject);

Tips

Add a CSS class to the HTML div and position your notifications on the page in a prominent location, e.g. 
<div class="growl" data-sys-growl></div>

.growl {
  position: fixed;
  width: 400px;
  top: 80px;
  right: 40px;
  z-index: 1000;
}

Notes

The contents of messageObject in the example, used to $emit the message should be the following attributes:

type: string

Values of string should relate to bootstrap alert types; info (default), success, warning, danger

title: string

Any plain text title to display in the growl alert

content: HTML

HTML content top be used in the alert

e.g.

var messageObject = {
      type: 'info',
      title: 'Example Growl',
      content: 'My <em>message</em>'
    };

Example

Click the button to add a growl message (limited to display of 3 messages)
