Angular Active Navbar Directive Highlight the current page from you navigation

Directive Overview

<nav data-sys-active-navbar>
    <a data-route="[uri-expression]">Link Name</a>
</nav>

data-sys-active-navbar Directive initialisation, put this at the highest point in your navigation, the 'active' class will propagate from the anchor tag to this tag in the html tree.

data-route="[uri-expression]" this is a regular expression to match when changing the route or $location.path() value in your single page application and should sit as part of the anchor tag to match from.

Demo

HTML 
<nav data-sys-active-navbar>
    <ul class="nav navbar-nav">
        <li>
            <a data-route="/path/to/page">Link Name</a>
        </li>
        <li>
            <a class="dropdown-toggle" data-route="/path/from.*">Link Name</a>
            <ul class="dropdown-menu">
                <li>
                    <a data-route="/path/from/page">Link Name</a>
                </li>
                <li>
                    <a data-route="/path/from/another/page">Link Name</a>
                </li>
            </ul>
        </li>
    </ul>
</nav>
 JavaScript 

Nothing required, just include the library in your application

Tips

This directive also has a listener for events sent to _pageScroll, this means it is possible to automatically scroll the page when it is loaded by adding the following to your controller:


.controller('scrollCtrl', ['$scope', '$location', function ($scope, $location) {
    if ($location.path()) {
        $scope.$emit('_pageScroll', $location.path().substring(1), 70);
    }
}])
Note that there are two parameters here the string identifier and an integer for the required offset on the page
